# Family Hub

This folder is ready to publish with GitHub Pages.

## Upload steps

1. Go to GitHub and create a new repository, for example `family-hub`.
2. Upload these files:
   - `index.html`
   - `manifest.json`
   - `icon.svg`
3. In the repository, go to **Settings → Pages**.
4. Under **Build and deployment**, set:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
5. Click **Save**.
6. Open the GitHub Pages URL in Safari on your iPhone or iPad.
7. Tap **Share → Add to Home Screen**.

## Note

This version stores data locally on each device. Hosting fixes the Safari/local-file issue and allows Home Screen installation, but it does not yet sync data between devices.
